# Privacy Policy for AI Tab Manager

Last Updated: January 2025

## Overview

AI Tab Manager is committed to protecting your privacy. This privacy policy explains how our Chrome extension handles your data.

## Data Collection and Usage

### What We Collect
- **Tab Information**: URLs, titles, and metadata of open tabs (processed locally)
- **User Settings**: Your preferences including API keys, theme settings, and categorization preferences
- **Tab Categories**: The AI-generated categories for your tabs

### What We DON'T Collect
- Personal identification information
- Browsing history beyond current open tabs
- Passwords or sensitive form data
- Analytics or tracking data

## Data Storage

### Local Storage Only
All data is stored locally on your device using Chrome's storage API:
- Saved tabs and categories
- User preferences and settings
- API keys (stored securely in Chrome's local storage)

### No Remote Servers
- We do not operate any servers
- No user data is transmitted to us
- All processing happens locally in your browser

## Third-Party Services

The extension communicates only with the AI service providers you configure:
- **Claude (Anthropic)**: When using Claude for tab categorization
- **OpenAI**: When using GPT models for tab categorization
- **Google AI**: When using Gemini for tab categorization

These communications:
- Only include tab titles and URLs for categorization
- Are initiated only when you click "Categorize Tabs"
- Use your own API keys
- Are subject to the respective AI provider's privacy policy

## Data Security

- API keys are stored locally and never transmitted to our servers
- All data remains on your device
- No analytics or telemetry is collected

## Your Rights

You can:
- View all stored data in the extension's settings
- Delete saved tabs at any time
- Clear all extension data by uninstalling the extension
- Choose which AI provider to use or disable AI features entirely
