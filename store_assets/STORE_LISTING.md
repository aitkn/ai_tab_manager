# Chrome Web Store Listing Information

## Extension Name
AI Tab Manager

## Short Description (132 characters max)
Intelligently categorize and manage your browser tabs using AI. Save important tabs, close unnecessary ones, and stay organized.

## Detailed Description

### Overview
AI Tab Manager uses artificial intelligence to help you organize your browser tabs efficiently. With just one click, it analyzes all your open tabs and categorizes them into three groups: Important Links (to keep open), Save for Later (to bookmark), and Can Be Closed (safe to close).

### Key Features
🤖 **AI-Powered Categorization**
- Intelligent tab analysis using Claude, OpenAI GPT, or Google Gemini
- Customizable AI prompts for personalized categorization
- Support for multiple AI providers with your own API keys

📑 **Smart Tab Management**
- Three-tier categorization system: Important, Save for Later, Can Be Closed
- Batch operations: save and close tabs by category
- Duplicate tab detection and management
- Group tabs by domain, date, week, or month

💾 **Save & Organize**
- Save tabs to a local database for later access
- Search through saved tabs
- Export saved tabs
- Track when tabs were saved and which AI model was used

🎨 **Modern Interface**
- Clean, Material Design-inspired UI
- Dark mode support (follows system theme)
- Responsive and fast
- Visual indicators for tab importance

🔒 **Privacy First**
- All data stored locally on your device
- No tracking or analytics
- Your API keys never leave your browser
- Open source for full transparency

### How It Works
1. Click the extension icon to open the popup
2. Configure your preferred AI provider and API key (one-time setup)
3. Click "Categorize Tabs" to analyze all open tabs
4. Review the AI's suggestions and make adjustments if needed
5. Save important tabs and close unnecessary ones with one click

### Perfect For
- Researchers managing multiple sources
- Developers with dozens of documentation tabs
- Students organizing study materials
- Anyone who tends to have too many tabs open
